# Contributing to Polystate Services

Thanks for your interest in contributing! This is a community-maintained directory of crypto and fintech tools for digital nomads. We value quality over quantity — if you've battle-tested a service, we want to hear about it.

## How to Contribute

### Suggest a Tool

1. [Open an Issue](https://github.com/Spider333/Polystate-Services/issues/new?template=suggest-tool.md) using the "Suggest a Tool" template
2. Or submit a Pull Request directly (see format below)

### Report a Dead Link or Outdated Info

1. [Open an Issue](https://github.com/Spider333/Polystate-Services/issues/new?template=report-dead-link.md) using the "Report Dead Link" template

### Submit a Pull Request

1. Fork the repository
2. Add your tool in the correct category section
3. Follow the entry format below
4. Submit a PR with a clear description

## Entry Format

Each tool entry in the community directory should follow this table format:

```
| # | Tool Name | One-line description of what it does | [domain.com](https://domain.com) | Status |
```

- **#** — Next sequential number in the category
- **Tool Name** — Official product name
- **Description** — One sentence, focus on what it does and why it's useful for crypto nomads
- **Link** — Official website URL only
- **Status** — Use `🆕` for new entries

## Quality Criteria

We accept tools that:

- Are **actively maintained** and operational
- Are **genuinely useful** for people living on crypto abroad
- Have a **working website** with clear product information
- Are **not scams** — we verify submissions before merging

We do NOT accept:

- Defunct or abandoned projects
- Tools with no clear use case for crypto nomads
- Duplicate entries (check existing list first)
- Promotional content or spam

## Affiliate Links Policy

- Community submissions should **not** contain affiliate/referral links
- Only maintainers may add affiliate links (marked with ↗)
- All affiliate relationships are disclosed transparently in the README
- Affiliate links support the Polystate platform at no extra cost to users

## Review Process

1. Submit your PR or Issue
2. A maintainer will review within 7 days
3. We may ask for clarification or personal experience with the tool
4. Once approved, the entry is merged into the directory

## Code of Conduct

Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before contributing.

## Questions?

Open an issue or reach out at [polystate.io](https://polystate.io).
